# A.L.I.C.E
一個當初為了[台姬殿](https://discord.gg/tvpc)設計的機器人\n
基於node.js跟discord.js所編寫\n
充滿大量寫死的UUID 不方便別人更改

## 主要功能
1. 關鍵字回覆
2. 身分組人數統計
3. 大規模訊息刪除
4. 給特定身分組的人 另一個身分組
5. 使用CSV做訊息量排序

## 可以優化的事項
